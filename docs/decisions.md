Business Decisions

Platform Type
- Multi-agent SaaS platform
- Single PWA

Authentication
- One login
- One user account
- One dashboard

Agent Model
- Each agent is independent
- Each agent has its own pricing
- Users can subscribe to multiple agents

Referral Model
- Referral rewards available
- Rewards are not cash
- Rewards do not reduce subscription fees

Points Model
- Points are marketing incentives
- Points unlock extra benefits
- Points never replace subscription payments

Revenue Rule
- Business profitability comes first
- Rewards must generate significantly more revenue than cost

Development Order
1. Platform foundation
2. SEO Agent
3. Lead Finder Agent
4. Outreach Agent
5. Additional agents
