AgentPlatform

A multi-agent SaaS PWA platform.

Goals:
- One login for all agents
- Free tier on every agent
- Separate subscription for each agent
- Referral and loyalty system
- Users can subscribe to one or multiple agents
- PWA installable on desktop and mobile

Initial agents:
1. SEO Agent
2. Lead Finder Agent
3. Outreach Agent
4. Mockup Agent

Long-term goal:
Build a marketplace of specialized business agents.
